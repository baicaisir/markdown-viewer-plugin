# Contributing

Thanks for considering a contribution!

## Development

Requirements:

- JDK 17
- Maven 3.9+ (or use `./mvnw`)

Run a local Jenkins for manual testing:

```bash
./mvnw hpi:run
```

Run all tests:

```bash
./mvnw verify
```

## Commit Style

Use conventional-commit style messages:

- `feat: ...` new feature
- `fix: ...` bug fix
- `docs: ...` documentation only
- `test: ...` adding/adjusting tests
- `chore: ...` build / tooling

## Security

If you find a security issue, do **not** open a public issue. Email the maintainer privately.
